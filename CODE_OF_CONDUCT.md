# Code of Conduct

## Our Pledge

We expect all participants to help keep this project respectful, focused, and useful.

## Standards

Examples of positive behavior:
- Be respectful and constructive.
- Give actionable feedback.
- Keep discussions focused on the code and the project.
- Be patient with new contributors.

Examples of unacceptable behavior:
- harassment or abuse
- personal attacks
- discriminatory language or behavior
- deliberate disruption

## Enforcement

Maintainers may remove, edit, or reject contributions that violate this code of conduct.

## Scope

This code of conduct applies to project discussions, issues, pull requests, and any other project spaces.
