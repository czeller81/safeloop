# Safeloop Real-World Dogfood Plan

This plan is for local, real-project validation of Safeloop v0.7 without changing the product architecture.

## Section 1: PLOTS

### Goal
Validate Safeloop observability in a local AI-assisted application.

### Tasks
1. README update workflow
2. Small feature implementation
3. Steering comparison test

### Capture
- events
- decisions
- risks
- approvals
- token usage
- cost tracking
- readiness score

### Observe
- Goal Drift
- Steering Intelligence
- Release Readiness

### Success Criteria
- useful dashboard signals
- minimal noise
- actionable reports
